Trabalho desenvolvido ao longo da matéria de programação I. Trata-se uma API REST uma plataforma voltada para o gerenciamento e resolução de questões de concursos públicos, permitindo que estudantes, professores e administradores tenham uma experiência completa na prática e análise de provas
anteriores.


A solução foi modelada para servir como estudo de caso acadêmico e como projeto real
de aplicação web com arquitetura moderna baseada em Spring Boot 3, JPA, Lombok,
H2/SQL Server, Swagger e muito mais.
